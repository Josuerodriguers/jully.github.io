# 💖 Nossa História

Um projeto web interativo desenvolvido com **React + TypeScript + Vite**, criado para representar visualmente uma linha do tempo de relacionamento com contador em tempo real, galeria de fotos e interações românticas 💕

---

## ✨ Funcionalidades

* ⏳ Contador de tempo juntos (dias, horas, minutos e segundos)
* 🖼️ Carrossel de fotos responsivo
* 🔍 Visualização ampliada (lightbox com fundo embaçado)
* ❤️ Efeito de chuva de corações ao clicar no botão
* 📱 Layout totalmente responsivo (mobile + desktop)
* 🎨 UI com efeito glassmorphism e gradiente moderno

---

## 🚀 Tecnologias utilizadas

* ⚛️ React
* 🟦 TypeScript
* ⚡ Vite
* 🎨 CSS moderno (gradientes, blur, animações)

---

## 📦 Como rodar o projeto

Clone o repositório:

```bash
gh repo clone Josuerodriguers/jully.github.io
```

Acesse a pasta:

```bash
cd seu-repo
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Abra no navegador:

```
http://localhost:5173
```

## 📁 Estrutura do projeto

```
src/
 ├── assets/        # imagens

```

---

## 💡 Destaques técnicos

* Carrossel adaptável a imagens verticais e horizontais (`object-fit: contain`)
* Lightbox com `backdrop-filter` (blur real no fundo)
* Sistema de animação dinâmica de elementos (chuva de corações)
* Layout flexível com `clamp()` para responsividade avançada

---

## ❤️ Interações especiais

Ao clicar no botão de coração:

* 💕 Uma animação de chuva de corações é disparada
* 🎯 Cada coração possui posição, tamanho e movimento aleatórios
* 🌟 Efeito visual leve e performático usando CSS + React
